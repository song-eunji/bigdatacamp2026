# 활동 기록 및 포트폴리오 생성 PRD

- 문제: 캠프·부트캠프와 같은 장기간의 활동에서 기록을 남기지 않으면 나중에 포트폴리오 작성 시 기억이 나지 않아 결국 대충 작성하게 된다.

- 타깃: 여러 날짜에 걸친 팀 프로젝트/캠프에 참여 중인 학습자  

- 유저 스토리: 캠프 참가자로서 매일 짧게 활동을 기록하면 그게 자동으로 쌓여서 마지막 날 포트폴리오 초안이 완성되길 원한다.

- 핵심 기능: 
  1. 매일 3줄 활동 로그 입력 → 자동 누적 
  2. 버튼 한 번으로 누적 로그를 근거로 성장 스토리형 포트폴리오 초안 생성 
  3. (wow) 하루 3줄씩 4일 입력하면 10초 안에 성장 스토리 초안 완성

- 비목표: 
  - 회원가입/로그인
  - 디자인 템플릿 커스터마이징
  - PDF 내보내기
  - 타인과의 공유/협업
  - 활동 로그 자동 요약 기능 (사용자가 작성한 로그를 기반으로 자동으로 요약 제공)
  - 리마인더 기능 (매일 활동 기록을 입력할 수 있도록 알림 제공, 범위 번짐 방지)


- 성공 기준: 
  - 로그 입력 → 저장까지 1초 안에 완료 
  - 4일치 로그로 10초 안에 포트폴리오 초안 생성 
  - 데모에서 5명 중 3명이 '이 초안 그대로 쓸 만하다'고 답한다
